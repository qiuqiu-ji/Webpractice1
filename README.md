# 班级网页项目

这是一个简单的班级展示网页，用于展示班级照片和同学信息。项目使用纯HTML5和CSS3开发，具有响应式设计和美观的动画效果。

## 功能特点

- 班级集体照展示
- 同学个人信息卡片
- 响应式设计，适配各种设备
- 美观的动画效果
- 简洁易用的界面

## 页面结构

### 顶部区域
- 班级名称标题
- 班级集体照片展示

### 学生信息区
- 网格布局的学生卡片
- 每张卡片包含：
  - 学生照片
  - 姓名
  - 爱好
  - 特长

## 使用说明

1. 添加照片：
   - 将班级集体照命名为 `class-photo.jpg` 放入项目根目录
   - 将学生照片命名为 `student1.jpg`、`student2.jpg` 等放入项目根目录

2. 添加学生信息：
   - 在 `index.html` 中复制 `student-card` 结构
   - 修改照片路径和学生信息
   - 注意修改 `--card-index` 值，确保动画效果正确
